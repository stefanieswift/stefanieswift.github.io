Arrays and Hashes

which should I use?

January 28

Arrays

An array is a collection of items that have an index number.

text

Hashes

Hashes are a collection of key and value pairs. A good time to use a hash instead of an array is if each key has multiple values. For instance let's create both an array and a hash for animals.

Animal Hash

Animals = Hash.new Animals = { 'dogs' => 'Golden Retriever, Cairn Terrier, and Pittbull', 'cats' => 'Tiger, Lion, and Kitty', 'whales' => 'Humpback, Orca, and Gray' }

Animal Array

Animals = array.new Animals = {cats, dogs, whales, tiger, lion}

As you notice you can still put all of the animals in an array or hash, but you do not have the ability to sort the animals as nicely in an array.
