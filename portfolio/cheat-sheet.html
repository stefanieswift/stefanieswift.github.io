  1. SELECT Statements
    • SELECT * FROM t
    • SELECT c1,c2 FROM t
    • SELECT c1,c2 FROM t WHERE conditions ORDER BY c1 ASC/DESC
    • SELECT DISTINCT c1,c2 FROM t

  2. Update Table Statements
    • INSERT INTO t (c1,c2…) VALUES (v1,v2...)
    • INSERT INTO t1 (c1,c2…) SELECT c1,c2… FROM t2 WHERE conditions
    • UPDATE t SET c1 = v1, c2 = v2,… WHERE conditions
    • DELETE FROM t WHERE conditions
    • TRUNCATE TABLE t

  3. Operators
    • SELECT * FROM t WHERE c1 [NOT] BETWEEN v1 AND v2
    • SELECT * FROM t WHERE c1 [NOT ] IN (v1,v2,…)
    • SELECT * FROM t WHERE c1 > v1 AND c1 < v2
    • SELECT * FROM t WHERE c1 = v1
    • SELECT * FROM t WHERE c1 <> v1

  4. Join Statements
    • SELECT * FROM t1 INNER JOIN t2 ON conditions
    • SELECT * FROM t WHERE c1 [NOT ] IN (v1,v2,…)
    • SELECT * FROM t1 INNER JOIN t2 ON conditions WHERE conditions
    • SELECT * FROM t1, t2 WHERE conditions
    • SELECT * FROM t1 LEFT JOIN t2 ON conditions
